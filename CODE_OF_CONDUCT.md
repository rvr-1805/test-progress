# Code of Conduct

## Our Commitment

We want this project to be welcoming and respectful for everyone.

## Expected Behavior

- Be respectful and constructive
- Give clear, actionable feedback
- Assume positive intent
- Help others learn

## Unacceptable Behavior

- Harassment, insults, or discrimination
- Personal attacks
- Spam or disruptive comments

## Collaboration Rule

Disagreements are normal. Keep discussions focused on ideas, not people.
